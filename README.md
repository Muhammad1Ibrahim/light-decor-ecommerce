# Light Decor — E-Commerce Website

A fully functional e-commerce store built for a real family lighting retail business with physical stores in Rawalpindi and Islamabad, Pakistan.

**Live site:** https://lightdecor.khingars.com

## What Was Built

A complete WooCommerce store covering:

- **Product catalogue** across 5 categories: Chandeliers, Wall Lights, Hanging Lights, Ceiling Lights, Garden Lights
- **Product pages** with PKR pricing, sale prices, and add-to-cart functionality
- **Full checkout flow** — cart, checkout, order management via WooCommerce
- **Repair Service page** — additional service offering beyond product sales
- **About Us & Contact pages** with real store locations (Raja Bazaar, Rawalpindi & Jinnah Super Market, F-7, Islamabad)
- **Newsletter signup** with discount incentive
- **Customer accounts** — login, order history via WooCommerce My Account

## Process

The project wasn't just building — it involved research and planning first:

- **Competitor SEO audit** using SEMrush — ran a site audit on a competing lighting retailer (thelightstore.pk), identifying technical issues including broken internal links, broken images, and duplicate meta descriptions that gave Light Decor an opportunity to outperform them on search
- **Customer persona development** — built from in-store observation of real customers
- **UI mockups** designed in Figma before development began
- **Sitemap and navigation planning** using draw.io
